## Libraries

- [Dagger](http://square.github.io/dagger/)

- [Picasso](http://square.github.io/picasso/)
- [OKHttp](http://square.github.io/okhttp/)
- [Retrofit](http://square.github.io/retrofit/)
- [GSON](https://github.com/google/gson)
